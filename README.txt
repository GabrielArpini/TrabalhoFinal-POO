Trabalho final de Programação Orientada a Objetos

Tema: Controle Financeiro

Grupo: Anna Giulia Rezende Santos, Gabriel Santos Arpini e Sabrina Santiago de Oliveira.

Observação:
O programa deve ser executado em modo administrador, a fim de permitir a criação e remoção de arquivos no computador, as escolhas de design foram feitas com o intuito de facilitar o entendimento do comportamento da aplicação.
O código está disponível no GitHub: https://github.com/GabrielArpini/TrabalhoFinal-POO/tree/main

Como abrir?
Basta entrar na pasta "Executavel" e abrir o arquivo "Setup" ele irá realizar as instalações necessárias e abrir o aplicativo, caso apareça mensagens de alerta do sistema operacional é necessário que continue mesmo assim.
